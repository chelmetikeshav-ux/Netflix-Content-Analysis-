# Netflix-Content-Analysis-
# 🎬 Netflix Data Analysis (Python)

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a Netflix dataset to understand trends in movies and TV shows available on the platform. The analysis focuses on exploring the dataset, cleaning missing values, and visualizing important insights related to Netflix content.

The project demonstrates the use of **Python data analysis libraries such as Pandas, Matplotlib, and Seaborn** to analyze real-world datasets.

---

## 📂 Dataset
The dataset used in this project is **netflix_titles.csv**.  
It contains information about Netflix content including:

- Title
- Type (Movie / TV Show)
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

---

## 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed

### 1. Data Exploration
- Displayed the first and last rows of the dataset
- Checked dataset shape and column names
- Examined data types of each column

### 2. Data Cleaning
- Identified duplicate records
- Checked missing values in the dataset
- Calculated total null values per column

### 3. Data Visualization
- Heatmap to visualize missing values
- Count plots to analyze distribution of Movies vs TV Shows

### 4. Content Analysis
- Searched for specific titles
- Analyzed release year trends
- Compared number of Movies and TV Shows available on Netflix

---

## 📈 Key Insights

- Netflix has a larger number of **Movies compared to TV Shows**
- Some columns such as **director, cast, and country contain missing values**
- Content releases have increased significantly in recent years
